細部の比較に最適。ピクセル単位で同じ位置を比較できる。

+ 細部の差分比較に最強 / ピクセル単位で一致する位置を比較
- 画像サイズが違うとズレる / ドラッグ操作が必要

B. 左右並列

before/after を横に並べる。全体の印象の違いを一目で把握。
スマホだと各画像が小さくなるのが弱点。

+ 全体比較が一目でわかる / 操作不要
- スマホで各画像が小さい / 細部比較は難しい

C. タップ切替

画面タップで before/after を瞬時に切り替え。
フルサイズで表示しつつ、パラパラ漫画のように差分が見える。

+ フルサイズ表示 / 直感的 / 差分が「動き」で見える
- 同時比較はできない / 記憶に頼る部分がある

提案: 全部入りタブ切替

上のワイヤーフレームの通り、比較モード内でタブで切替可能にする。
子画像一覧から任意の子をタップ → 比較モードに入る。
用途に応じてスライダー/並列/タップ/透過を選べる。
最後に使ったモードを localStorage で記憶。
