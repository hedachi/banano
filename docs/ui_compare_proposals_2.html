B. 上下並列

縦持ちの画面を活かして上下に配置。
左右並列よりも各画像が大きく表示でき、同時に見られる。
スクロールでピンチズームとも相性がよい。

+ 両方同時に見える / 各画像が画面幅フル / 操作不要
- 各画像の高さが半分になる / 横長画像向き

C. タップ切替

画面タップで before/after を瞬時に切り替え。
画面幅いっぱい、高さもたっぷり使える。
パラパラ漫画のように差分が「動き」で見える。

+ 最大サイズで表示 / 差分が動きで直感的にわかる / 片手操作
- 同時には見られない / 微妙な差は記憶頼み
